# Super Store Sales Dashboard

This project presents a comprehensive **Sales Analysis Dashboard** for a Super Store built using **Power BI** .

### Project Overview
The dashboard provides deep insights into the store's sales performance, profitability, and customer behavior. Data was extracted, cleaned, and analyzed using SQL, and an interactive dashboard was developed in Power BI to help visualize key business metrics effectively.

### Key Metrics
- **Total Sales**: $1.57 Million
- **Total Quantity Sold**: 22K+
- **Total Profit**: $175.26K
- **Average Delivery Rate**: 3.76 days

### Analysis Included
- Sales and Profit breakdown by **Category** (Furniture, Office Supplies, Technology)
- Regional performance (West, East, Central, South)
- Sales by **Ship Mode** and **Payment Mode**
- Top performing **Sub-Categories** (Phones, Chairs, Binders, etc.)
- Monthly sales trend analysis
- Customer segment analysis

### Tools & Technologies Used
- **Power BI** – Interactive dashboard and visualizations
- **DAX** – Custom measures and KPIs

### Dashboard Features
- Clean and user-friendly interface
- Multiple interactive filters
- Key Performance Indicators (KPIs)
- Various charts including Pie, Bar, and Trend Line

### Future Scope
- Sales forecasting
- Customer segmentation
- Year-over-Year performance comparison
